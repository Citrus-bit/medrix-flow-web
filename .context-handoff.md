# MedrixFlow Web — Project Context

## Overview

Promotional landing page for [MedrixFlow](https://github.com/Citrus-bit/medrix-flow), a full-stack AI agent orchestration platform built on LangGraph. Single-page glassmorphism website with medical teal theme, Chinese-facing.

**Dev server**: `pnpm dev` → `http://localhost:5173` (port chosen to avoid medrix-flow's 1000/2024/3000/8001)

## Tech Stack

- Next.js 16.2.1 (App Router, Turbopack)
- React 19, TypeScript 5
- Tailwind CSS 4 (CSS-first config via `@theme inline` in globals.css)
- `motion` (framer-motion v12) — import from `"motion/react"`
- `lucide-react` — all icons (no GitHub icon in this version, using inline SVG)
- `clsx` + `tailwind-merge` → `cn()` in `src/lib/utils.ts`

## Design System

### Colors (from source project's `frontend/src/styles/globals.css`)

Light mode:
- Background: `#f8fffe` | Foreground: `#0f2b2e`
- Primary: `#0891b2` (medical teal) | Primary fg: `#ffffff`
- Secondary/Accent: `#e0f7fa` | Secondary fg: `#0e4a5a`
- Muted: `#eef8f9` | Muted fg: `#4a7c82`
- Border: `#c8e6e9`

Dark mode (CSS ready, no toggle):
- Background: `#0b1e21` | Foreground: `#e0f2f4`
- Primary: `#22d3ee` | Card: `#0f2930` | Border: `#1c3f45`

Key gradients:
- `.medrix-gradient`: `linear-gradient(135deg, #0891b2 0%, #06b6d4 40%, #14b8a6 100%)`
- `.gradient-text`: teal → cyan → cyan-500 on text

### Glassmorphism Classes (defined in globals.css)

| Class | Use |
|-------|-----|
| `.glass` | Basic: 8% white bg, 16px blur |
| `.glass-light` | Cards: 65% white, 20px blur, subtle shadow |
| `.glass-strong` | Navbar/overlays: 85% white, 24px blur |
| `.glass-card` | Interactive cards: 70% white, hover lifts -2px |
| `.medrix-glow` | Teal box-shadow glow |
| `.section-divider` | Horizontal gradient line between sections |

All glass classes have `.dark` variants.

### Typography

- Headings: Space Grotesk (300–700)
- Body: DM Sans (300–700)
- Code: JetBrains Mono (400–500)
- Base font-weight: 300 (light)

### Animations (Tailwind `@theme inline`)

`animate-float`, `animate-glow-pulse`, `animate-shimmer`, `animate-fade-up`, `animate-fade-in`, `animate-scale-in`, `animate-slide-in-left`, `animate-slide-in-right`

## Page Sections (scroll order)

1. **Navbar** (`navbar.tsx`) — Fixed `top-4 left-4 right-4`, transparent → glass-strong on scroll. Links: #features, #architecture, #quickstart, #download. GitHub button with inline SVG icon.

2. **Hero** (`hero.tsx`) — min-h-screen, 3 floating gradient orbs, badge pill, "AI 超级代理系统" h1, terminal mockup showing clone+config+install+dev flow, 4 stats cards (17 middleware, 3 sub-agents, 20+ tools, 277 tests).

3. **Features** (`features.tsx`, `#features`) — "为什么选择 MedrixFlow", 7 cards in 3-col grid (multi-agent, sandbox, memory, streaming, zero-config UX, multi-channel, security audit). Each has: icon, title, scene tag, description, highlight badges.

4. **Highlights** (`highlights.tsx`) — "深入了解", 3 alternating left/right sections with terminal demos:
   - "17 层中间件流水线" — middleware chain architecture
   - "记忆不是缓存" — LLM-driven memory with correction detection
   - "子代理不是噱头" — parallel task decomposition

5. **Architecture** (`architecture.tsx`, `#architecture`) — "系统架构", compact 4-column service cards (Nginx:1000→LangGraph:2024→Gateway:8001→Frontend:3000) + routing table with descriptions.

6. **Installation** (`installation.tsx`, `#quickstart`) — "快速开始", prerequisites badges (Python/uv/Node.js/pnpm/nginx), 4-step first-time install timeline, "日常启动" section with `make dev`, "常用命令" reference grid.

7. **Download** (`download.tsx`, `#download`) — "获取 MedrixFlow", git clone with copy, GitHub repo link, ZIP download, Docker deploy option (`make up`), expanded tech stack badges (LangGraph/LangChain/FastAPI/Next.js/React/TailwindCSS/Shadcn UI/TanStack Query/LangGraph SDK), MIT badge.

8. **Footer** (`footer.tsx`) — Logo, description, links (GitHub/配置指南/架构详解/API参考/Issues), tech badges, copyright.

## Key Patterns

- All client components use `"use client"` directive
- Motion animations use `whileInView` for scroll triggers with `viewport={{ once: true }}`
- Copy-to-clipboard: tries `navigator.clipboard` first, falls back to textarea+execCommand
- GitHub icon is an inline SVG component (lucide-react v1.7.0 doesn't export a GitHub icon)
- Background: 4 fixed blurred gradient orbs in `page.tsx` provide depth for glassmorphism
- No dark mode toggle (light mode only, but CSS supports dark)
- `prefers-reduced-motion` respected in globals.css
