# MedrixFlow Web

[MedrixFlow](https://github.com/Citrus-bit/medrix-flow) 官方宣传网站。

## 技术栈

- **Next.js 16** (App Router, Turbopack)
- **React 19** + **TypeScript 5**
- **Tailwind CSS 4** (CSS-first `@theme inline` 配置)
- **motion** (Framer Motion v12) — 滚动动画
- **lucide-react** — 图标库

## 开发

```bash
pnpm install
pnpm dev        # http://localhost:5173
```

## 构建

```bash
pnpm build
pnpm start
```
